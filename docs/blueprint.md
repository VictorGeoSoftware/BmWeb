# **App Name**: PriceWise

## Core Features:

- Login/Authentication: Secure user authentication with username and password validation.
- Home Screen with Menu: A home screen that provides a navigation menu on the left.
- PDF Upload: Allow users to upload PDF documents containing electricity prices using a drag-and-drop area.
- Price Proposal Upload Button: A button to confirm and finalize the electricity price proposal upload.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey trust and sophistication.
- Background color: Light gray (#F5F5F5), a desaturated near-tone to the primary.
- Accent color: Teal (#009688), an analogous tone to the primary hue, with brighter saturation and lightness, for interactive elements.
- Font: 'Inter', a grotesque-style sans-serif font with a modern, neutral look suitable for both headlines and body text.
- A left-side navigation menu for key functions, and a drag-and-drop area for uploading PDF documents prominently displayed on the main content area.
- Simple, clear icons for menu options and file upload indicators.
- Subtle transitions and feedback animations during file uploads.
